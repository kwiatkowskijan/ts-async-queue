# ts-async-queue

## Async Local Memory Queue in TypeScript

### Requirements
- [Node.js](https://nodejs.org/en/) 12.0.0 or newer

### Run Program

1. Clone the repository:
   ```sh
   git clone <git lab template url>
   ```

2. Navigate to the project directory:
   ```sh
   cd ts-async-queue
   ```

3. Install dependencies:
   ```sh
   npm install
   ```

4. Build and run:
   ```sh
   npm start
   ```
